# Feature Flags with Environment Variables

Control feature activation and behavior using environment variables.

## Usage

Different environments can have different feature flags:

- **Development**: All features enabled, higher limits
- **Production**: Conservative settings, lower limits

## Benefits

- ✅ Enable/disable features without code changes
- ✅ Different settings per environment
- ✅ Easy A/B testing
- ✅ Gradual feature rollouts
