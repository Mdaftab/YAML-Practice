# Local Development Setup

Configuration optimized for local development with sensible defaults.

## Quick Start

```bash
# Copy example
cp .env.local .env

# Edit for your setup
nano .env

# Start local services (Docker)
docker-compose up -d

# Run application
npm start
```

## Features

- ✅ Defaults for local development
- ✅ Mock API mode enabled
- ✅ Debug logging
- ✅ Hot reload enabled
- ✅ Skip authentication option
